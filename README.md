# מערכת שכר חכמה – Payroll Management System

**Smart Payroll Management System – Designed for small and medium businesses**  
A modern, efficient, and fully automated payroll solution that simplifies employee management, salary calculations, tax reporting, and monthly/yearly documentation.  
Built for reliability, accuracy, and ease of use — without the need for heavy or expensive enterprise systems.

מערכת שכר מתקדמת לניהול עובדים, הפקת טפסים, חישובי שכר, ניכויים, הפקת קבצי דיווח, ניהול תיקיות, ושידור נתונים למוסדות.  
המערכת מיועדת לעסקים קטנים ובינוניים, ומאפשרת עבודה פשוטה, מהירה ויעילה ללא צורך במערכות כבדות או יקרות.

---

##  פיצ'רים מרכזיים

- ניהול עובדים מלא (פרטים אישיים, שכר, ניכויים, שעות)
- הפקת טפסים רשמיים: 102, 126, 106 ועוד
- יצירת קבצי XML לשידור למוסדות
- ניהול תיקיות אוטומטי לכל עובד ולכל חודש
- חישובי שכר אוטומטיים (מס הכנסה, ביטוח לאומי, פנסיה)
- הפקת דוחות חודשיים ושנתיים
- שמירה אוטומטית של נתונים וקבצים
- ממשק משתמש ידידותי ותמיכה מלאה בעברית

---

##  התקנה

1. הורדת קבצי הפרויקט מהריפו.
2. התקנת התלויות:

   ```bash
   pip install -r requirements.txt
   ```

3. הפעלת המערכת:

   ```bash
   python main.py
   ```

---

## שימוש בסיסי

לאחר הפעלת המערכת ניתן:

- להוסיף ולנהל עובדים  
- להפיק טופס 102 לחודש נבחר  
- להפיק טופס B102 , H102 שנתי חודשי  
- ליצור קבצי XML לשידור למוסדות  
- לנהל תיקיות עובדים וקבצי שכר לפי חודשים ושנים  
- להפיק דוחות שכר ודוחות סיכום
